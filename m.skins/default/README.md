m_bootstraps
============

모바일 부트스트랩 출석부 스킨

개요
----

이 소프트웨어는 '모바일 부트스트랩 출석부 스킨'입니다.

이 소프트웨어는 GPL v2을 채택한 오픈 소스 소프트웨어로써 자유롭게 사용할 수 있는 자유 소프트웨어 입니다.
이 라이선스 조건을 준소하는 조건하에 누구나 자유롭게 사용할 수 있습닏나.

주요 기능
---------
* XE 출석부 소프트웨어에서 기본적으로 지원하는 모바일 스킨을 좀 더 개량된 버전으로 쓸 수 있습니다.
